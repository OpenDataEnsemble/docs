# Try ODE Online

Experience ODE without installation through our interactive demo and hosted examples.

## 🚀 Quick Demo

### Live Demo Server
Try ODE right now in your browser:

**🔗 [Demo Server](https://demo.ode.dev)**

**Demo Features:**
- ✅ Pre-configured server
- ✅ Sample forms and data
- ✅ Mobile app simulation
- ✅ Admin portal access
- ✅ No registration required

### What You Can Do
1. **Explore Forms**: Browse sample data collection forms
2. **Test Data Entry**: Submit test observations
3. **View Synchronization**: See data sync in action
4. **Try Admin Portal**: Manage users and settings
5. **Experience Custom Apps**: See custom application examples

## 📱 Mobile App Demo

### Android Demo
**🔗 [Download Demo APK](https://demo.ode.dev/formulus-demo.apk)**

**Features:**
- ✅ Pre-configured demo server
- ✅ Sample forms included
- ✅ Offline data collection
- ✅ Synchronization testing
- ✅ Custom application examples

### Installation
```bash
# Install via ADB (if you have Android Debug Bridge)
adb install formulus-demo.apk

# Or download directly to your Android device
# Enable "Install from unknown sources" in settings
```
