# About

Support links, license information, and project context for ODE Desktop.

Open **Data management → About** (`#/data/about`).

## About ODE Desktop

ODE Desktop is part of **Open Data Ensemble** — an offline-first ecosystem for field data collection, synchronization, and stewardship. It helps you manage observations locally and develop app bundles before deploying to Formulus in the field.
